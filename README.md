# Airpost Dynamic Caption Placement (Demo)

This repository showcases a **public demo** of my work on the *Dynamic
Caption Placement (DCP)* system developed at
[Airpost.ai](https://airpost.ai). The goal of this project is to
intelligently place captions in videos by analyzing object saliency and
scene composition.
